# JSON解析网站需求文档

## 项目概述
开发一个纯前端的JSON解析工具网站，提供JSON格式验证、格式化、压缩等常用功能，所有功能在浏览器端完成，无需后端支持。

## 功能需求（第一版）

### 核心功能
1. **JSON解析验证**
   - 验证输入的JSON格式是否正确
   - 错误时显示具体的错误位置和原因

2. **JSON格式化**
   - 将压缩的JSON格式化成带缩进的易读格式
   - 支持自定义缩进空格数（2个或4个空格）

3. **JSON压缩**
   - 将格式化的JSON压缩成单行，去除多余空格和换行

4. **字符串转义/去转义**
   - 转义：将JSON字符串中的特殊字符进行转义处理
   - 去转义：将转义后的字符串还原

5. **Unicode转换**
   - Unicode转中文：将\uXXXX格式的Unicode编码转换为中文字符
   - 中文转Unicode：将中文字符转换为\uXXXX格式的Unicode编码

6. **本地缓存**
   - 自动保存当前编辑内容到浏览器localStorage
   - 页面刷新后自动恢复上次编辑的内容
   - 提供清除缓存功能

### 界面设计要求
- **交互友好**：操作简单直观，按钮功能明确
- **布局清晰**：左右分栏布局（输入区|输出区）
- **美观简洁**：现代化的UI设计，响应式布局

## 技术方案

### 技术栈
- **前端框架**：React 18 + TypeScript
- **构建工具**：Vite
- **UI组件库**：Ant Design
- **代码编辑器**：Monaco Editor（VSCode同款）
- **样式方案**：CSS Modules 或 styled-components

### 项目结构
```
JsonRun/
├── src/
│   ├── components/          # 组件
│   │   ├── JsonEditor/     # JSON编辑器组件
│   │   ├── FunctionPanel/  # 功能按钮面板
│   │   └── ErrorDisplay/   # 错误显示组件
│   ├── utils/              # 工具函数
│   │   ├── jsonUtils.ts    # JSON处理函数
│   │   ├── unicodeUtils.ts # Unicode转换函数
│   │   └── storageUtils.ts # 本地存储函数
│   ├── hooks/              # 自定义Hooks
│   ├── types/              # TypeScript类型定义
│   └── App.tsx             # 主组件
├── public/                 # 静态资源
└── package.json           # 项目配置
```

### 核心实现思路
1. **JSON处理**：使用浏览器原生`JSON.parse()`和`JSON.stringify()`
2. **本地存储**：使用`localStorage`进行数据持久化
3. **编辑器**：使用Monaco Editor提供语法高亮和错误提示
4. **布局**：采用左右分栏的响应式布局

### 部署方案
- **静态部署**：构建后生成静态文件，可部署到GitHub Pages、Vercel等平台
- **无需服务器**：纯前端项目，不需要后端服务器支持
